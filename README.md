# E-Learning-Platform# 
A lightweight, responsive E-Learning platform featuring course filtering, interactive UI, and specialized learning paths. Built with HTML, CSS, JavaScript, and Bootstrap.

## ✨ Key Features
- **Dynamic Filtering:** Filter courses by category (Development, Design, etc.).
- **Fully Responsive:** Works perfectly on Mobile, Tablet, and Desktop.
- **Bootstrap 5 UI:** Modern components for better user engagement.

## 🚀 How to Run Locally
1. Clone the repo: `git clone https://github.com/YOUR_USERNAME/REPO_NAME.git`
2. Open `index.html` in your browser.
